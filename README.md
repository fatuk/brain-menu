# Brain-like 3d menu

## [Demo](https://fatuk.github.io/brain-menu/public)

## How to run?

* download this repo
* `npm install && bower install`
* for develop run `gulp`
* for build run `gulp build`
* open in browser `http://localhost:3000/`

## To remove images:

`sed 's/"url.*/===/g' <brain-v1.json >new-brain.json`
